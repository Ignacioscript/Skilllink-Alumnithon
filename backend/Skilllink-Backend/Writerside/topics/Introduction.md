# Introduction

## 🧠 SkillLink – Collaborative E-learning Platform

## 🧩 What is SkillLink?

**SkillLink** is a web-based collaborative learning platform that connects learners and mentors through code challenges, personalized mentorship, and real-time communication.

It’s designed to help people grow technically by learning through action, guided by mentors, and supported by a developer community.

---

## 🎯 Objective

To create a space where users can:

- Improve their technical skills through real-world challenges
- Receive personalized mentorship
- Collaborate with others in the community
- Track their learning journey and progress

---

## 👥 Who is it for?

- **Learners:** Users who want to improve their skills, take on challenges, and receive feedback.
- **Mentors:** Experienced users who offer mentorship, evaluate challenges, and support others.
- **Admins:** Users who manage platform content, users, and moderation.

---

## 🔑 Core Features

- 🔐 Secure login and registration with JWT
- 👤 User profiles with bio, skills, and social links
- 🔎 Mentor and collaborator search by skill or interest
- 🧠 Mentorship system (request, accept, reject sessions)
- 🧪 Interactive challenges with code editor (CodeMirror/Monaco)
- 💬 Real-time chat using WebSockets
- 📊 Evaluations and feedback on submissions
- 🔔 Notification system
- 🧾 Certifications and course structures (optional module)

---
